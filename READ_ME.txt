SOFTWARE REQUERMENT- PYTHON 3.6 OR ABOVE,SQLITE3,
TO SEE THE DATABASE YOU HAVE TO DOWNLOAD -DB BROWSER FOR SQLITE.

#================================================================

To run this file(REGISTRATION FORM) through command prompt. JUST COPY THE 
PATH OF THE FILE AND PASTE IN COMMAND PROMPT THEN ENTER THE FILE NAME WITH THE
EXTENTION OF .PY 

ABOUT SOFTWARE - IN A SEARCH-BAR OF REGISTRATION WINDOW YOU HAVE TO ENTER 
THE STUDENT ID . 
* TO SEE THE FEE DETAILS OF THE STUDENT JUST ENTER THE STUDENT ID(BY VIEWING ON
DATABASE) THEN HIT LOGIN (IN FEE DETAILS WINDOW).
*TO ADD THE FEE ENTER IT ON SECOND ENTRY AND PRESS ADD FEE BUTTON.
*DELETE- ENTER THE STUDENT ID IN SEARCH-BAR AND PRESS DELETE BUTTON 
*UPDATE- FIRSTLY SEARCH THE STUDENT PROFILE ON SEARCH-BAR THEN MAKE CHANGE 
IN ANY ENTRY(EXCEPTION- FEE) AND PRESS UPDATE BUTTON
